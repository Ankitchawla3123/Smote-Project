# SMOTE Project

This project focuses on handling imbalanced datasets using different **SMOTE (Synthetic Minority Over-sampling Technique)** methods and evaluating their impact on machine learning models.

## Project Files

### 1. SmoteTechniques.ipynb
This notebook contains the implementation and visualization of different **SMOTE techniques**.  
It includes graphs and analysis demonstrating how each sampling method balances the dataset and affects data distribution.

### 2. acmproject.ipynb
This notebook contains the **dataset loading, preprocessing, model training, and evaluation**.  
Different machine learning models are applied to the dataset to compare performance before and after applying SMOTE techniques.

### 3. acm.py
module for data pre processing pipeline and applying models

## Objective
The main objective of this project is to study how different **SMOTE-based resampling techniques** help improve model performance when dealing with **imbalanced datasets**.
